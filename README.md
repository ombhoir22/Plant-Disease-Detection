# 🌱 Plant Disease Detection using AI

## 📌 Description
This project detects plant diseases using deep learning (MobileNetV2).

## 🚀 Features
- Upload leaf image
- Predict disease
- Show confidence score
- Suggest treatment

## 🛠 Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- Flask

## 📂 Dataset
PlantVillage Dataset

## ▶️ How to Run
```bash
cd app
python app.py
